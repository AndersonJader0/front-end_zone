Arquivo HTML é apenas um arquivo de texto, mas não um comum. É um arquivo de texto que contém as marcações HTML que dão significado ao conteúdo. Título Subtítulo Parágrafo Lista Máquinas não sabem identificar como nós humanos o que é um título, parágrafo ou lista como surge através de um livro ou jornal. É necessário colocar marcações (TAGS) de inicío e fim nesses conteúdos.
